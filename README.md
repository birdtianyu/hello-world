# 如何理解概率论中的“矩”？
https://www.matongxue.com/madocs/412.html

数学期望是一阶原点矩（表示分布重心）

方差是二阶中心距（表示分布对重心的离散程度）

偏态是三阶中心矩（表示分布偏离对称的程度）

峰态是四阶中心距（描述分布的尖峰程度，例如正态分布峰态系数=0）
